Youdao.alfredworkflow
======================

有道翻译 workflow for Alfred v2

默认快捷键 "f"，多个翻译结果，回车或直接点击将结果复制到剪切板。

通过输入"ff" 即可以在有道词典的页面中查询输入的单词。方便用户查看更详细的信息以及将单词加入生词本操作。
